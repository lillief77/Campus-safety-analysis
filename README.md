# NYC-Drinking-Fountains
Dataset and methodology for drinking fountain access in NYC perception story 

# Overview 
- This data exposes which burroughs are provided less free drinking water fountains
- This data examines  if the descrepincies between each of the burroughs' access to drinking fountains has anything to do with their overall wealth/status 

## Data Sources
- This data was provided by NYC parks, through the agency of parks and recreation. (3,850 fountains recorded)

## Methodology
- Added pivot tables to identify trends in the dataset.
- Cleaned empty spaces around any data
  
## Limitations 
- Data was sampled for surface-use. Not very in depth.
- Does not show the population size of each burrough
-  Does not show when the fountains were put in to use

## How to Use
- Use this dataset to compare trends of free water in the different New York City burroughs
- See the different places these drinking fountains are most common, point out trends (positive or negative)
